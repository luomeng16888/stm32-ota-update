# \# 基于STM32的嵌入式AI模型权重远程OTA更新系统

# 

# \## 项目简介

# 

# 面向边缘AI设备模型迭代更新的需求，以STM32F407微控制器为核心，设计并实现了一套支持差分更新、断点续传与版本回滚的轻量级远程OTA系统，并配套开发了基于Flask的云端管理平台。（因为bsdiff+deflate格式板子运行不稳定，目前的代码以简单差分格式为主）

# 

# \## 核心功能

# 

# \- \*\*双模式更新：\*\* 支持全量更新和基于BSDiff差分算法的增量更新，差分更新平均节省约70%传输数据量

# \- \*\*断点续传：\*\* 基于HTTP Range请求的断点续传机制，网络中断后自动恢复下载

# \- \*\*安全Bootloader：\*\* STM32片内1MB Flash四分区设计（Bootloader + APP\_A/APP\_B + 参数区），A/B交替更新与失败自动回滚

# \- \*\*云端管理平台：\*\* 基于Flask开发，支持设备注册与心跳维护、固件/模型版本管理、用户认证及更新策略动态决策

# \- \*\*模型热切换：\*\* GRU时序预测模型经INT8量化后权重仅1520字节，通过X-CUBE-AI指针替换机制实现运行时模型热加载

# 

# \## 系统架构

# 

# ┌──────────────┐    WiFi/HTTP    ┌──────────────────────┐

# │  ESP8266     │   JSON + 分块    │   SQLite + Web后台    │

# │  SD卡 + LCD  │   下载          │   BSDiff差分生成      │

# └──────────────┘                 └──────────────────────┘

# 

# \## 硬件平台

# 

# \- \*\*主控：\*\* STM32F407ZGT6（1MB Flash, 192kB SRAM）

# \- \*\*WiFi模块：\*\* ATK-MW8266D (ESP8266)

# \- \*\*存储：\*\* Micro SD卡（8GB，SDIO 4位总线）

# \- \*\*显示：\*\* 2.8寸TFT-LCD（320x240，FSMC驱动）

# \- \*\*开发板：\*\* 正点原子STM32F407探索者V3

#
